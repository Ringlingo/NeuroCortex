## 术语表

| 术语 | 解释 |
|------|------|
| EDA | 探索性数据分析，了解数据分布和特征 |
| Overfitting | 模型在训练集上表现好，测试集上差 |
| Cross-validation | 交叉验证，评估模型泛化能力 |
| Feature Engineering | 特征工程，提取有意义的特征 |

## 数据清洗 Checklist

- [ ] 缺失值处理（删除/填充/预测）
- [ ] 异常值检测（箱线图/Z-score/IQR）
- [ ] 数据类型转换（日期/类别/数值）
- [ ] 重复行去除
- [ ] 格式统一（大小写/单位）

## 模型选择指南

| 任务类型 | 推荐算法 |
|---------|---------|
| 分类 | 随机森林、XGBoost、逻辑回归 |
| 回归 | 线性回归、XGBoost、LSTM |
| 聚类 | K-Means、DBSCAN、层次聚类 |
| 降维 | PCA、t-SNE、UMAP |

## 可视化最佳实践

- **分布**：直方图、箱线图、KDE
- **关系**：散点图、热力图、pairplot
- **时序**：折线图、面积图
- **分类**：饼图、条形图、堆叠图
